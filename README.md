Interactive Profile Card

It is a JavaScript project that displays interactive profile cards.

Profile Card Features:

1. Three interactive profile cards
2. Personalized greeting message.
3. Independent click counter for each of the card
4. DOM manipulation using JavaScript
5. Template literals for dynamic text.
6. Arrow function for formatting greetings
7. Responsive card design with CSS
8. Button and card hover effect.
